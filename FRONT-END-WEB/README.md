# SEMAPAM Water Admin

Base Angular 17 para el portal administrativo de control de abastecimiento de agua.

## Incluye
- login JWT con `/login` y `/me`
- layout administrativo con sidenav y topbar
- módulos lazy: dashboard, usuarios, productos, zonas, abastecimientos, tickets, arqueo, reportes y configuración
- servicios desacoplados y tipado `ApiResponse<T>`
- base Mapbox Draw para zonas
